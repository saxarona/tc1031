\section*{Iteración}

Considera una función \texttt{pueblo(n)} que imprime $n$ veces el mensaje ``Hola pueblo'' usando un \textbf{for}.
Con esta información, responde a las siguientes preguntas:

\begin{itemize}
	\item ¿Cuántas operaciones \textit{exactamente} hace la función si la llamo con \texttt{pueblo(15)}?
	\item ¿Cuántas operaciones \textit{exactamente} hace la función si la llamo con \texttt{pueblo(7)}?
	\item ¿Puedes hacer una función para definir su complejidad? Si sí, escríbela.
	\item En caso de que pudieras hallar una función, ¿en qué \textbf{orden} de magnitud común está? Escribe tu respuesta usando $\mathcal{O}, \Theta$ y $\Omega$.
\end{itemize}
